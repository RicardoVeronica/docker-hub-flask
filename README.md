# This image will help you to start with Python Flask Microframework

### Just pull the image
```
$ docker pull ricardoveronica/flask-app:1.0.0
```

### Run the image
```
$ docker-compose run --rm ricardoveronica/flask-app:1.0.0
```

### Open your browser in
0.0.0.0:5000

### Start to code

## Happy coding
